# Prompt Slot System (Anti-Drift Standard)

Use this slot order exactly to improve reproducibility.

## Canonical Slot Order

1. **Subject Core** — who/what is depicted
2. **Pose & Camera** — angle, crop, posture
3. **Art Nouveau Line Language** — whiplash, contour rhythm, silhouette behavior
4. **Motif Stack** — dominant/support/micro motif families
5. **Frame Architecture** — halo/arch/cartouche/border structure
6. **Palette Script** — 3–5 core hues + 1 accent rule
7. **Surface Finish** — lithographic grain, paper tone, edge behavior
8. **Typography Reservation** — if needed, blank title/cartouche region
9. **Quality Anchors** — coherence, period plausibility, readability
10. **Output Constraint** — aspect ratio / format

## Canonical Skeleton

`[SUBJECT CORE], [POSE & CAMERA], Art Nouveau with flowing whiplash curves and controlled contour hierarchy, dominant motifs [DOM], support motifs [SUP], micro motifs [MICRO], [FRAME ARCHITECTURE], restrained historical palette ([COLOR SET]) with selective accents [ACCENT RULE], lithographic print texture and period paper tone, [TYPOGRAPHY RESERVATION], high compositional coherence and period plausibility, [ASPECT RATIO/FORMAT].`

## Negative Slot (Short)

`photorealism, modern UI/iconography, neon/chrome effects, hard cinematic lighting, anime/manga stylization, cluttered ornament, low-detail anatomy`

## Rapid Patch Add-Ons

- **If too busy:** `reduced ornament density, clearer negative space, stronger silhouette separation`
- **If too flat:** `foreground vine layer, midground figure emphasis, halo depth separation`
- **If too modern:** `turn-of-the-century print character, matte inks, period-compatible decorative forms`
- **If border weak:** `three-band border architecture with outer restraint line, ornamental middle band, inner breathing margin`
