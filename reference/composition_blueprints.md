# Composition Blueprints (Numeric Guides)

These blueprint percentages provide repeatable layout control.

## 1) Portrait Poster Blueprint (2:3)

- Top title zone: **12%** height
- Primary figure zone: **62%** height
- Lower motif + info zone: **18%** height
- Outer margins: **4%** each side
- Inner breathing margin: **3%**

### Focal Rules
- Eye line sits around **34–40%** from top.
- Halo diameter: **30–42%** of composition width.
- Dominant ornament should not exceed **35%** of area outside figure silhouette.

## 2) Full Figure Blueprint (3:4)

- Header ornament/text zone: **10%**
- Full figure vertical run: **70%**
- Base pedestal/cartouche: **14%**
- Top/bottom breathing bands combined: **6%**

### Focal Rules
- Head top margin: minimum **5%** from top boundary.
- Feet/base contact should align within bottom **12–16%**.

## 3) Circular Medallion Blueprint (1:1)

- Outer ring thickness: **8–12%** radius
- Inner ornament ring: **6–10%** radius
- Image core: remaining center radius

### Focal Rules
- Face center offset from exact center: max **±4%**.
- Tangent clearance between hair and inner ring: **1.5–3%** radius.

## 4) Triptych Blueprint (Vertical)

- Width split: **24% / 52% / 24%**
- Gutter between panels: **2–3%** total width each
- Shared title band across all panels: **10–14%** height
- Shared footer band: **8–12%**

### Focal Rules
- Central panel carries strongest contrast and largest figure.
- Side panels should operate at **70–85%** visual intensity of center.
