# Art Nouveau Style Repository

> *"To every age its art, to every art its freedom."*  
> — Vienna Secession motto, inscribed on the Secession building, 1898

A comprehensive reference repository documenting the Art Nouveau movement (c. 1888–1914) in its full historical, aesthetic, technical, and contemporary dimensions — and providing production-ready tools for creating Art Nouveau-inspired images with consistent visual language.

---

## Repository Purpose and Scope

This repository serves two complementary functions:

**1. Historical and aesthetic documentation:** A thorough scholarly reference covering the movement's origins, regional variants, key practitioners, canonical works, technical methods, visual principles, and contemporary influence. Useful for researchers, designers, artists, and students seeking to understand Art Nouveau in depth.

**2. Production tools for AI image generation:** A complete toolkit — prompt templates, style specifications, motif dictionaries, composition blueprints, and QA workflows — for generating Art Nouveau-inspired images with AI tools, maintaining the movement's specific visual language with reproducible accuracy.

**Coverage spans:**
- Historical period: c. 1848 (Pre-Raphaelite precedents) to present
- Geography: France, Belgium, Germany, Austria, Scotland, Catalonia, Italy, Netherlands, United States
- Media: Poster art, painting, architecture, glass, jewellery, furniture, ceramics, textiles, typography, digital illustration, tattoo art
- Key figures: 15+ historical practitioners profiled; contemporary practitioners included

## Visual Style Overview

Art Nouveau (c. 1888–1914) is characterised by:

- **Organic naturalness** — all forms derived from living organisms; no straight lines where curves are possible
- **The whiplash curve** — the defining formal signature; sinuous, energetic S-curves and spiraling tendrils
- **Asymmetry** — dynamic compositional balance rather than bilateral mirroring
- **Flat colour planes** — influenced by Japanese ukiyo-e; strong outline with even colour fill, no chiaroscuro
- **Integrated ornament** — decoration grows from structure; borders are part of the composition
- **The female figure** — the "New Woman" as primary figurative subject; flowing hair as ornamental line element
- **Botanical vocabulary** — iris, water lily, wisteria, poppy, dragonfly, peacock feather as recurring motifs
- **Muted warm palette** — antique gold (`#C9A84C`), warm cream (`#F5E6C8`), dusty rose (`#D4A0A0`), muted sage (`#8A9E78`)

**Regional flavours:**
| Variant | Character | Key Figures |
|---|---|---|
| French/Belgian | Sinuous, warm, figurative | Mucha, Toulouse-Lautrec, Horta |
| German Jugendstil | Earthy, botanical, typographic | Eckmann, Behrens, Olbrich |
| Vienna Secession | Geometric-organic, black-gold-white | Klimt, Hoffmann, Moser |
| Glasgow Style | Rectilinear, austere, lilac-grey | Mackintosh, M. Macdonald |
| Catalan Modernisme | Organic, structural, ceramic | Gaudí, Domènech i Montaner |

## Notes on Repository Organisation

This repository integrates two distinct types of content:

1. **Historical documentation** (the `history/`, `characteristics/`, `color-palettes/`, `typography/`, `examples/`, `influences/`, `practitioners/`, `architecture/`, and `reference/` directories) provides scholarly reference material.

2. **Production tools** (`STYLE_SPEC.md`, `PROMPT_PACK.md`, `MOTIF_DICTIONARY.md`, the `prompts/` and `reference/` production files, `QA_WORKFLOW.md`, `QA_PATCH_LIBRARY.md`) provide practical tools for AI-assisted image creation.

The historical documentation informs and validates the production tools: the palette system is grounded in actual historical palettes; the motif dictionary reflects authentic Art Nouveau ornamental vocabulary; the style specification is based on documented visual principles.

Cross-references throughout the documentation link between these two layers, so that production decisions can always be traced back to historical sources.
