# Summary: Design Grammar of Art Nouveau and “Le Style Mucha”

## Core focus
This source appears to define the visual grammar of Art Nouveau—especially the Mucha variant—covering line behavior, composition, symbolic motifs, palette, and figure treatment.

## Key guidance extracted
- **Signature linework:** prioritize flowing “whiplash” curves, asymmetrical rhythm, and organically branching forms.
- **Figure + frame integration:** place idealized feminine figures inside ornamental structures (arches, halos, botanical borders, medallions).
- **Nature-derived patterning:** repeat florals, vines, seedpods, leaves, feathers, and radial sunburst motifs.
- **Decorative flattening:** prefer poster-like plane clarity over photographic depth; stylization is more important than realism.
- **Period-appropriate palette:** muted jewel/earth harmonies with controlled contrast and gold-like accents.
- **Typography + ornament coupling:** letterforms should feel integrated with borders and visual motifs, not overlaid.

## Practical implications for generation
- Start with a **curvilinear scaffold** before rendering details.
- Reserve geometric hard edges for **framing accents**, not dominant shapes.
- Keep textures subtle (lithographic/grain cues) to preserve period feel.
- Use modern elements only if fully translated into Art Nouveau ornament language.
