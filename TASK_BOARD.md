# Task Board — Incremental Repo Expansion

Use this board to ship enhancements in small, focused pushes.

## How to Use
1. Pick exactly one `P0` or up to two `P1` tasks.
2. Implement only that scope.
3. Run QA checks.
4. Commit + note what changed.
5. Move to next task.

---

## P0 (Core Fidelity)

- [ ] **T-001 Prompt Slot Standard**
  - Add strict prompt slot order and examples.
  - **DoD:** A contributor can build prompts without style drift.

- [ ] **T-002 Typography System Spec**
  - Define title/subtitle/caption hierarchy, spacing, and ornament anchoring.
  - **DoD:** Text-first poster prompts produce stable, period-compatible layouts.

- [ ] **T-003 Composition Blueprint Percentages**
  - Add zone maps for portrait, full panel, triptych, medallion formats.
  - **DoD:** Designers can place focal elements using numeric guides.

- [ ] **T-004 Figure Stylization Ratios**
  - Add head/body ratios, neck elongation limits, hand pose archetypes.
  - **DoD:** Figure proportions are intentional and repeatable.

- [ ] **T-005 Failure Patch Library**
  - Add symptom → prompt patch snippets.
  - **DoD:** Common failures can be corrected in one iteration.
